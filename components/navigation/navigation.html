Navigation

Navigation bar is used to navigate through the web apps or websites. They can be in horizontal or vertical shape.

Below is an example for horizontal navigation bar.

LOGO
  • HOME
  • CONTACT
  • ABOUT
    •             
          <nav class="navigation-hz">
            <div class="left-nav">
              <a href="#" class="nav-logo">LOGO</a>
            </div>
            <div class="right-nav">
              <li><a class="link" href="#">HOME</a></li>
              <li><a class="link" href="#">CONTACT</a></li>
              <li><a class="link" href="#">ABOUT</a></li>
            </div>
          </nav>

    Below is an example for vertical navigation bar.

    • LOGO
    • HOME
    • CONTACT
    • ABOUT
                
          <nav class="navigation nav-vr">
            <ul class="nav-ul">
              <li><a class="active link" href="#">LOGO</a></li>
              <li><a href="#" class="link">HOME</a></li>
              <li><a href="#" class="link">CONTACT</a></li>
              <li><a href="#" class="link">ABOUT</a></li>
            </ul>
          </nav>
