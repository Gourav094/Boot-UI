List

Lists can be both spaced and stacked in nature.

Below is an example for spaced stack that are normal lists.

  • Item 1
  • Item 2
  • Item 3
  • Item 1
  • Item 2
  • Item 3
            
             <ul class="list">
              <li>Item 1</li>
              <li>Item 2</li>
              <li>Item 3</li>
             </ul>
             <ul class="list list-inline">
              <li>Item 1</li>
              <li>Item 2</li>
              <li>Item 3</li>
             </ul>
