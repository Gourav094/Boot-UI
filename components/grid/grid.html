Grid

Grid is used to build 2 dimensional layouts. You can have control over both column and grid at a same time.

Below is an example for 2 items in a grid.

Item-1
Item-2
            
          <div class="grid-layout grid-2column">
            <div class="box box-1">Item-1</div>
            <div class="box box-2">Item-2</div>
          </div>

Below is an example for 3 items in a grid.

Item-1
Item-2
Item-3
            
          <div class="grid-layout grid-3column">
            <div class="box box-1">Item-1</div>
            <div class="box box-2">Item-2</div>
            <div class="box box-3">Item-3</div>
          </div>
